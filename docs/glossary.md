# 용어 매핑표 — 어려운 말 → 쉬운 말

> 중학생이 이해할 수 있는 표현으로 바꾸는 기준표.
> 모든 문서에 동일하게 적용한다.

## 핵심 개념

| 어려운 말 | 쉬운 말 | 비고 |
| --- | --- | --- |
| Anti-Sovereign Design | 반주권 설계 | 공식 영문명 |
| 락인 (lock-in) | 갇힘 | "한 번 들어가면 나오기 힘든 상태" |

| 탈출권 | 빠져나갈 권리 | |

| 판별식 | 갇힘 확인표 | 4가지 항목 체크 |
| 합작 락인 | 짝꿍 갇힘 | "둘이 합쳐서 만드는 갇힘" |
| 포크 (fork) | 갈라타기 | "같은 것을 복사해서 따로 운영" |
| 포크 비용 비대칭 | 갈라탈 때 드는 힘 차이 | "회사는 쉽고 사용자는 어려움" |

| 관계적 락인 | 관계 갇힘 | "돌봄처럼 사람 사이 관계로 생기는 갇힘" |
| 인정 체계 | 인정 시장 | "학력처럼 누군가가 인정해주는 체계" |
| 대리 발언권 | 대신 말해주는 권리 | |
| 강제의 역설 | 강제 딜레마 | "풀어주려다 더 묶이는 딜레마" |
| 포획 속도 | 갇히는 속도 | |
| 창발 (emergence) | 교차점에서 생김 | "각자 따로 움직이다 만나서 생기는 현상" |

| 위계 | 차례 | "누가 위고 누가 아래인지" |

| 예외 모드 | 비상 모드 |
