# 반주권 설계 (Anti-Sovereign Design)

> 주권을 없앤다고 말하면 거짓이다. 주권이 굳는 속도를 늦추고,
> 문을 열어두고, 사람들이 더 나은 방을 직접 만들어 옮겨갈 수 있게 하는 설계다.

이 문서는 "반주권 설계" v4 프레임워크를 정의한다. 이 프레임워크는
국가·지역·플랫폼·회사·병원·학교·가족 등 모든 권력 구조에 적용 가능한
락인(lock-in) 기반 주권 분석 도구다.

## 사례 검증 현황

다섯 사례가 프레임워크의 보편성을 검증한다 — 지역 주권, 비지역 주권, 물리적
주권, 관계적 주권, 인정 체계 주권에 같은 판별식이 적용되며, 합작 락인 구조가
다섯 영역에서 동일하게 발생한다.

| 사례 | 주권 유형 | 락인의 본체 | 검증된 것 |
| --- | --- | --- | --- |
| E-7 비자 | 지역적 (영토 기반) | 체류 허가 + 고용 결합 | 고용주-국가 합작 락인, 락인 열지도, 처방의 행정표 전환 |
| 플랫폼 계정 정지 | 비지역적 (네트워크 기반) | 계정 + 네트워크 효과 | 영토 없는 주권, 계정=디지털 시민권, 포크 비용 비대칭, 4개 합작 구조 |
| 의료 기록 이전 | 물리적 (인프라 기반) | 물리 인프라 + 기록 | 데이터 이동성 3층 부재, 물리 인프라 포크 불가, 5개 합작 구조, 정보 비대칭 |
| 돌봄 의존 | 관계적 (관계 기반) | 관계 + 대체 돌봄 부재 | 관계적 락인, 탈출 피해, 대리 발언권, 탈출권 확장(대체 생존조건 동반), 5개 합작 구조 |
| 학력 인증 독점 | 인정 체계 (인정 기반) | 인정 독점 + 입시 트랙 | 제도적 이동성 극단 사례, 삼각 관계적 락인, 인정 시장 다원화, 5개 합작 구조 |

### 다섯 사례가 공통으로 입증한 것

1. **판별식이 다섯 가지 주권 유형에 모두 적용** — 지역, 비지역, 물리적, 관계적, 인정 체계
2. **합작 락인의 4단계 패턴** — 각 주체는 자기 역할만 수행, 교차점에서 창발, 처방은 동시 풀어야, 시행 주체 역설 반복
3. **락인 위험도의 연속성** — 같은 구조에서 사용자/환자/돌봄/학생 유형에 따라 위험도가 다름
4. **데이터 이동성 3층이 진짜 병목** — 기술적 이동성만 충족, 의미·제도적 이동성 부재
5. **부문별 포크 원칙이 다르게 적용됨** — 디지털(진짜 포크), 의료(인정+접근), 물리(접근 보장만), 관계(대체 돌봄 인프라), 인정(인정 다원화)
6. **탈출권은 맥락에 따라 확장됨** — 단순 이탈 → 데이터 동반 → 인정 동반 → 대체 생존조건 동반 → 인정+트랙 보호
7. **강제의 역설이 모든 영역에서 반복** — 처방을 채택할 권한을 가진 주체가 처방에 의해 자신의 권한이 줄어드는 구조

## 핵심 한 줄

**어떤 권력도 사람의 생존조건을 쥔 채 설명 없이 묶어둘 수 없다.**
락인 위험은 계속 측정되어야 하고, 그 측정 권력은 분리되어야 하며,
방어장치는 서로 다른 시간표로 작동해야 한다.
시스템은 하나의 선을 향해 통치되는 게 아니라,
탈출 가능한 실험들이 공개 비교되며 개선되어야 한다.

> 국가 커널이 예외상태로 전환될 때 나타나는 최고강도 모드다. 17장이 16장의 상위가
> 아니라, 16장이 17장의 예외 모드다.

## 탈출권의 진화 — 5사례 비교

| 사례 | 탈출권의 형태 | 핵심 요구 |
| --- | --- | --- |
| E-7 | 비자에서 빠져나갈 권리 | 단순 이탈 |
| 플랫폼 | 데이터를 가져갈 권리 | 이탈 + 데이터 |
| 의료 | 기록을 이전할 권리 | 이탈 + 기록 + 인정 |
| 돌봄 | 대체 생존조건 동반 이탈 | 이탈 + 대체 돌봄 + 방치 방지 |
| 교육 | 인정이 이전되는 이탈 | 이탈 + 기록 + 인정 + 입시 트랙 보호 |

탈출권이 갈수록 요구하는 것이 많아진다. 이건 프레임워크가 "탈출권만 있으면
된다"에서 시작해서, 실제 적용하면서 **탈출권이 맥락에 따라 다른 무게를
가짐**을 발견한 과정이다.

## 지적 계보

이 프레임워크의 조각은 기존 이론에 있다. 독특한 것은 조각 자체가 아니라,
**락인 판별식 하나로 국가·플랫폼·회사·병원·학교·가족을 같은 테이블에 올리는 방식**이다.

- Hirschman — exit-voice-loyalty: 출구가 쉬워야 정치가 개선된다
- Nozick — 최소국가: 국가 기능의 최소화
- Rawls — 무지의 베일: 최악의 위치에 있는 사람 기준으로 설계
- Mouffe — 적대적 민주주의(agonistic democracy): 합의가 아니라 분쟁의 가시화
- Srinivasan — Network State: 영토 없는 주권
- Hohfeld — 권리-의무 대응: 권리에는 강제 주체가 필요하다는 역설
- BFT 합의 — 분산 시스템에서 신뢰할 수 없는 노드가 임계값을 넘으면 합의가 깨짐
- 오픈소스 포크 — 포크 위협이 규율 작용을 함
- 의학 IRB — 사람을 대상으로 하는 실험에는 사전 동의와 중단 기준이 필요함
